FLARE Network

Follow the below instructions to mint The Phoenix Token (PHX) on FLARE Network.

Summary

The Phoenix Token (PHX) is a deflationary ERC20 token on the Flare Network. It serves as a utility token and an open-source smart contract, enabling FLR holders to actively contribute to the growth and sustainability of the Flare Network.

One of the key principles behind the design of The Phoenix Token is its fair launch. There is no pre-mined allocation of PHX with total supply initialized at 0 upon deployment. This ensures that no individual or member of the project team has an unfair advantage, mitigating concerns related to price manipulation or centralized control.

The minting process of PHX is closely tied to the burning of FLR, the native token of the Flare Network. Participants who wish to mint PHX can do so by burning their FLR through The Phoenix contract. The ratio is set at fixed 1:1, meaning that for every FLR burned, an equal amount of PHX is minted and issued to the participant.

STEP 1:

Obtain Flare (FLR)

To participate in the minting process of Phoenix (PHX) Tokens the Networks native currency Flare (FLR) is required. If you do not currently hold any FLR, here are some methods to acquire them:

  • Centralized Exchanges such as Uphold or Bitrue (Note: KYC may be required).
  • Participation in the Flare Network Ecosystem - by delegating to the Flare Time Series Oracle.
  • Community Hosted Airdrops.

Already hold Flare (FLR) tokens? SKIP THIS STEP

STEP 2:

Create a Wallet

Before proceeding with the burning process, it is essential to ensure that you have a compatible wallet capable of supporting the Flare Network and FLR tokens. Here are some popular wallet options:

  • Metamask (Requires additional steps) - A widely used browser extension wallet that supports various blockchain networks, including the Flare Network. It offers a user-friendly interface, robust security features, and seamless integration with decentralized applications (dApps) built on the Flare Network.
  • Bifrost Wallet (Recommended) - A reputable wallet option with native Flare Network support! It is a multi-chain wallet that offers a range of features, including secure storage, easy token management, and support for interacting with decentralized finance (DeFi) applications on the Flare Network.
  • D'Cent Wallet (Requires additional steps) - A hardware wallet solution that provides enhanced security for managing your FLR tokens. It offers a user-friendly interface and supports multiple blockchain networks, including the Flare Network. With its offline storage and robust encryption features, D'Cent Wallet ensures the protection of your assets.

Already have a suitable wallet? SKIP THIS STEP

Add Flare Network & PHX Token

(For Metamask / D'Cent wallet users)

  1. Follow these instructions to Add Flare Network.
  2. Add The Phoenix Token (PHX): 0x06fb7579e28E1ca55AFA52272296a89a879d690a

STEP 3:

Initiate Burn Process

Using your wallet interface, initiate a transaction to send the desired amount of FLR tokens to The Phoenix Token (PHX) contract address. Be certain to double check the accuracy of the contract address before attempting to burn any FLR - You can do this by verifying the address using the Flare Network Explorer

The Phoenix Token (PHX) contract address: 0x06fb7579e28E1ca55AFA52272296a89a879d690a
